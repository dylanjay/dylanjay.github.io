Ray Tracer (C++)

Github

Description

Computer Graphics course project to implement a ray tracer that draws to the screen

Details

  • Used the GLUT openGL library
  • Modeled using flat shading, phong shading, addition of shadows, and reflections
  • Rays are shot recursively and bounced to simulate reflections
  • Image is anti-aliased by shooting multiple rays across the pixels
  • Rays hit objects and correspond to RGB values to be fed into a buffer to dynamically write to the screen
